🚀 Fonctionnalités incluses
Moodle officiel (moodlehq/moodle-php-apache:8.3)
Base de données MariaDB 11.4 avec persistance
Cache Redis pour les performances
Cron automatique pour les tâches Moodle
Reverse proxy Traefik avec SSL Let's Encrypt automatique
Sauvegardes quotidiennes vers Dropbox (base + fichiers)
Volumes persistants pour toutes les données importantes
📋 Prochaines étapes pour le déploiement
Dans Coolify :

Aller dans Applications → New Application
Choisir Docker Based → Docker Compose Empty
Sélectionner votre dépôt GitHub : https://github.com/fremar64/moodle-coolify-stack.git
Branche : main
Chemin : / (racine)
Lancer le déploiement
Configuration DNS :

Vérifier que ecole-en-ligne.ceredis.net pointe vers l'IP de votre serveur Coolify
Variables d'environnement :

Dans Coolify, configurer les variables selon vos besoins (mot de passe admin, token Dropbox, etc.)
Premier accès :

Une fois déployé, accéder à https://ecole-en-ligne.ceredis.net
Suivre l'assistant d'installation Moodle
🔧 Personnalisations possibles
Modifier les mots de passe dans les variables d'environnement
Configurer le token Dropbox pour les sauvegardes
Ajuster les ressources selon votre serveur
Ajouter des plugins Moodle via le volume moodle_html
Votre dépôt GitHub est maintenant prêt pour le déploiement avec Coolify ! 🎉
